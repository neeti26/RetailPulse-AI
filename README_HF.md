# 🏬 RetailPulse AI

Smart Mall Intelligence Agent built with **Google ADK**, **Gemini 2.5 Flash**, and **MongoDB Atlas MCP Server**.

## Setup

Add these secrets in Space Settings → Variables and secrets:
- `MONGODB_URI` — your MongoDB Atlas connection string
- `GOOGLE_API_KEY` — your Gemini API key
- `MALL_NAME` — e.g. `Sunrise Mall`

See the [GitHub repo](https://github.com/neeti26/RetailPulse-AI) for full setup instructions.
